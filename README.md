# Oloola

Oloola is an api to manage driver and trucks. 
It tracks trips made by trucks using google-maps-api.

##Technologies
- Spring boot
- Docker
- Gradle
- Postgresql
- Hibernate
- Hibernate-spatial


#How to run 
- Run ```./gradlew docker``` to build the docker image
- Run ```docker-compose up``` to run the docker application
